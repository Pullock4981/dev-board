Question-1: What are the different ways to select an element in the DOM?

1. By id. Like: document.getElementById("Id name")
2. By class. Like: document.getElementsByClassName("class name")
3. By tag name. Like: document.getElementsByTagName("Tag name")
4. By querySelector. Like: document.document.querySelector(".myClass")
5. By querySelectorAll. Like: document.document.querySelectorAll(".myClass")

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: It returns the HTML content of an element.
innerText: It returns the text content of an element.
textContent: It returns the text content of an element.

Question-3: What is event delegation in the DOM?

Event delegation is a technique involving adding event listeners to a parent element instead of adding them to the descendant elements. The listener will fire whenever the event is triggered on the descendant elements due to event bubbling up the DOM.
This is useful when you have a parent element with multiple children that you want to have the same event.
It is more efficient than adding an event listener to each child element.

Question-4: What is event bubbling in the DOM?

Event bubbling is a type of event propagation where the event first triggers on the innermost target element, and then successively triggers on the ancestors (parents) of the target element in the same nesting hierarchy till it reaches the outermost (top-level) element.
This is the default behavior of events in the DOM, but it can be stopped or prevented using event.stopPropagation() or event.preventDefault() methods.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element:
var element = document.createElement("div");
To add an element:
parentElement.appendChild(element);
To remove an element:
parentElement.removeChild(element);
